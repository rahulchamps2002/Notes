Midterm 1 Study Guide

  • Chapter 1: Descriptive Statistics
    • Pictoral Methods of Statistics:
      • Stem-and-Leaf Plot
      • Dotplot
      • Frequency and Relative Frequency Distributions for Discrete and Continuous Data
      • Histograms (Frequency, Relative Frequency) for Discrete and Continuous Data
    • Boxplot:
      • Find the 5-number summary (min, max, quartiles/fourths) and the interquartile range/fourth spread
      • Identify and Classify outliers (extremes or medium) in a data set (if any)
      • Construct a boxplot that indicates outliers (if any)
      • Answer questions concering comparative (side by side) boxplots. For Example:
        • Are the Medians approximately the same? Are the spreads (standard deviations) of the sets about the same? Any Outilers?
    • Numerical Methods of Descriptive Statistics:
      • Mean, trimmered mean, median
      • Range, sample standard deviation (s), sample variance (s2). Review both the definition formula and he short-cut formula for s2.
  • Chapter 2: Probability
    • Find the sample space and list outcomes of an event of an experiment.
    • Describe an event using set operations such as complementation, union, or intersection.
    • Determine whether given given events are mutually exclusively (pairwise disjoint).
    • Go over the axiom and properties of probabilities in Section 2.2
    • Review counting techniques and their applications to probability problems: the product rule, permutations, and combinations.
    • Conditional probability:
      • The definition
      • The Multiplication Rule
      • The Law of Total Probabilities
      • Bayes Theorm
      • Make sure you know how to use tree diagrams to solve a problem involving conditional probabilities.
    • Independent Events:
      • The Definition
      • The Multiplication rule of Independent Events
    • Review the definition of mutually independent events
  • Chapter 3: Sections 1-3: Discrete Random Variables
    • Find the set of possible value of a RV
    • Find the pmf and graph it (a line graph or probability histogram)
    • Find the cdf and graph it (there is a high chance of including the problem in the exam)
    • Compute the mean, variance, and standard deviation of a RV. Review the short-cut formula for the variance, that is, V(X) = E(X2) - (E(X))2.
    • Find the expected value of a function of a RV (E(h(X)) = ?).
    • Review the properties of the mean and variance of a linear function of a RV X. That is, E(aX + b) = ?, V(aX + b) = ?
